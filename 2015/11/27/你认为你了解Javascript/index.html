你认为你了解Javascript？

By 叶文翔

2015-11-27 更新日期:2015-11-27

文章目錄

这里有5个小脚本，有助于你真正理解JavaScript核心–闭包和作用域。没有在控制台运行之前，尝试回答每个案例中会弹出什么内容，然后你可以创建一个测试文件去核对你的答案。你准备好了吗？ 

1、

   1: if (!("a" in window)) {
   2:     var a = 1;
   3: }
   4: alert(a);
2、

   1: var a = 1,
   2:     b = function a(x) {
   3:         x && a(--x);
   4:     };
   5: alert(a);
3、

   1: function a(x) {
   2:     return x * 2;
   3: }
   4: var a;
   5: alert(a);
4、

   1: function b(x, y, a) {
   2:     arguments[2] = 10;
   3:     alert(a);
   4: }
   5: b(1, 2, 3);
5、

   1: function a() {
   2:     alert(this);
   3: }
   4: a.call(null);

我的预测答案分别是：undefined、1、不知道、10、null

答案在本文末尾，看答案之前，你敢留下你的猜测吗？

正确答案：1、undefined 2、1 3、function a(x){ return x * 2} 4、10 5、[object window]
